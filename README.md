# Running Industrial-database-app

### Install dependencies for ** node-server-app ** and ** vue-client-app ** by using :

```
npm install
```

### Running ** vue-client-app ** in dev mode

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
